# 📊 Inventory Management Dashboard

A professional Inventory Management Dashboard built using Excel and Power BI.

---

## Project Overview

This dashboard helps businesses monitor inventory performance using interactive visuals and KPIs.

---

## Tools Used

- Microsoft Excel
- Power BI
- DAX
- Power Query

---

## Dashboard Features

- Inventory Value
- Current Stock
- Purchase vs Sales Analysis
- Warehouse Analysis
- Product Category Analysis
- Inventory Turnover
- Days of Inventory
- KPI Cards
- Interactive Slicers
