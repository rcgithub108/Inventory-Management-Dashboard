# Inventory Management Dashboard

## Overview

This project is an Inventory Management Dashboard built using Microsoft Excel and Power BI.

## Tools Used

- Microsoft Excel
- Power BI
- DAX
- Power Query

## Features

- Inventory Value
- Current Stock
- Purchase Analysis
- Sales Analysis
- Warehouse Analysis
- Category Analysis
- Interactive Dashboard
